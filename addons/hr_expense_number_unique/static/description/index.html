Odoo App for Expense Number.

This app generates an auto unique sequence number for every Expense and Expense Report / Expense Sheet.

Main Features:

  • This app auto generates sequence numbers for expenses and expense sheets as shown below.
  • You can change prefixes as per your numbering pattern by going into the sequence menu.
