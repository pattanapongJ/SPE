Configuration

User Setting: Enable the' Cost Center' option.

In Invoice Configuration there is 'Cost Center' sub-menu and tree view.

'Cost Center' from view, where you can create a 'Cost Center'.

Sales

'Cost Center' field in sale quotation form and order line. If you change the cost center in the form then it will change in order line relatively.

Sales Analysis pivot report by 'Cost Center'.

Group sale orders by 'Cost Center'.

Group by 'Cost Center' in analysis view.

Account

'Cost Center' field in invoice form and invoice line.

Group invoices by 'Cost Center'.

'Cost Center' field in vendor bill form and bill line.

Group Vendor Bills by 'Cost Center'.

Invoice Analysis pivot report by 'Cost Center'.

Group by 'Cost Center' in analysis view.

Purchase

'Cost Center' field in the purchase order form and product line.

Group purchase order by 'Cost Center'.

Purchase Analysis pivot report by 'Cost Center'.

Group by 'Cost Center' in analysis view.

Expenses

'Cost Center' field in 'My Expenses'.

Group my expenses by 'Cost Center'.
