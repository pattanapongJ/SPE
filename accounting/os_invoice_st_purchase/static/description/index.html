Bill Payment Status In Purchase

Helps to see Payment Status of Bill in Corresponding PO.

Overview

This module brings an option to see the purchase order's invoice payment status in its corresponding purchase form view as well as tree view.

Configuration

No additional configuration required

Features

  • The Module adds functionality to see bill payment status of corresponding Purchase Order.
  • In Single Bill Case, module shows exactly the same status in purchase order that updated in bill.
  • In Multiple Bill Case, 'Paid' : If all bills are paid, 'partially paid' : if any of the bills are paid, 'in payment' : if any of the bills are in payment,'not paid' : if no bills are paid
