Compatibility

Community Enterprise Odoo.sh

Overview and Features

  • Manage multiple purchase teams
  • Add team to purchase orders
  • Restrict user access other team documents, only access own team documents
  • Team purchase on reporting

Configuration

Go to Apps, search and install module Purchase Team



Go to Purchase Team menu: Purchase > Configuration > Purchase Team
Create a new purchase team and add member to them



Set the user to group Purchase / User if only allowed to access his team's documents



How it works

Add representative user and team to Purchase Order



Restrict access to documents based on purchase team
"Purchase / User group" user session can read only his team's documents and the empty representative order



"Purchase / Administration group" user session access all of purchase documents



Allow to analyze the report based on purchase team



Disable restrict access purchase documents by archiving the following access rules
